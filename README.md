# Kommunaler Haushaltsassistent

Experimentelles Repository zur Vorbereitung eines KI-gestützten Assistenten für die Analyse kommunaler Haushaltspläne.
Das Projekt befindet sich derzeit in der Definitions- und Konzeptionsphase.

Der Haushaltsassistent wird mit realen, öffentlich zugänglichen Haushaltsplänen getestet.
Beispiel: Haushaltsplan Bad Soden 2026
(https://www.bad-soden.de/stadt/stadt-politik/politik-wahlen/haushaltsplaene/haushaltsplan-2026/
)

**Aus urheber- und lizenzrechtlichen Gründen werden keine Originaldokumente (PDF, Tabellen oder extrahierte Daten) in diesem Repository gespeichert.**
Stattdessen werden synthetische Testdaten und abstrahierte Beispielauszüge verwendet.
